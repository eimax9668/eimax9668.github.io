概要

View Transitions APIとは、Webページの画面遷移やDOMの状態変化を、 まるでネイティブアプリのようにスムーズで視覚的に連続性のあるアニメーションで表現するためのブラウザ標準APIです。

対応ブラウザ

View Transitions APIの対応ブラウザは2026年1月現在、Chrome/Edge 111以上（SPA用）、Chrome/Edge 126以上（MPA用）が安定対応です 。Safari 18以上、Firefox 144以上もSPAで対応が進んでいます。

ページ３

主な特徴とメリット

  • ブラウザ標準機能

    • 外部ライブラリに依存せず、ブラウザが直接アニメーションを処理します。

  • SPA・MPA対応

    • ページ全体のリロードなしで動作するSPA、および従来のページ遷移が発生するMPAのどちらにも適用可能です。

  • 高いパフォーマンス

    • ブラウザネイティブの機能で実行されるため、高速で滑らかなアニメーションが期待できます。
